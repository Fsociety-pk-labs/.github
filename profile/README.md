# FSociety PK Labs

> ⚡ A collection of cybersecurity challenges designed for beginners to practice, learn, and grow.

---

## About

**FSociety PK Labs** is a dedicated repository where we create and share hands-on cybersecurity challenges.

These labs are built to help beginners:
- Understand real-world vulnerabilities  
- Practice problem-solving skills  
- Gain practical cybersecurity experience  

---

## What You'll Find Here

🔹 Custom CTF-style challenges  
🔹 Web exploitation labs  
🔹 Cryptography challenges  
🔹 Networking-based tasks  
🔹 OSINT exercises  
🔹 Beginner-friendly scenarios

## Challenge Format

Each challenge includes:

- 📌 **Description**  
- 🏷 **Category**  
- 🎯 **Difficulty (Easy / Medium / Hard)**
